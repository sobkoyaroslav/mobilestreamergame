# Поднятие стримера в одиночку

Игра-кликер на Python + Tkinter. Тема — раскрутка канала стримера: выходишь
в эфир, набираешь фолловеров, прокачиваешь канал. Без JavaScript и без
сторонних библиотек — только стандартная библиотека Python.

Оформление выполнено в тёмной фиолетовой стилистике стриминговых сервисов.
Это намеренно дженерик-тема: в проекте нет чужих логотипов, названий и
точных брендовых цветов, поэтому вопросов по авторским правам не возникает.

## Как запустить игру на другом компьютере

Есть два способа. Самый простой — готовый `.exe`: он не требует
установленного Python и вообще ничего, кроме самого Windows.

### Способ 1. Готовый .exe (рекомендуется)

1. Возьми файл **`Поднятие стримера в одиночку.exe`** из папки `dist/`.
   Это всё — exe самодостаточный, остальные файлы копировать не нужно.
2. Перенеси его на другой компьютер (флешка, облако, почта — как удобно).
3. Положи exe в папку, где есть права на запись: рабочий стол, «Документы»
   или отдельная папка. **Не** запускай его из защищённых от записи мест
   (корень диска `C:\`, `C:\Program Files`) — рядом с exe игра создаёт
   файл сохранения `clicker_save.json` и звуки `snd_*.wav`.
4. Двойной клик по exe — игра запускается.

**Требования:** Windows 64-bit. Устанавливать Python и какие-либо
библиотеки не нужно.

**Предупреждение SmartScreen.** При первом запуске Windows может показать
синее окно «Система Windows защитила ваш компьютер». Так бывает с любой
новой программой без цифровой подписи. Нажми **«Подробнее»**, затем
**«Выполнить в любом случае»** — это безопасно.

### Способ 2. Из исходного кода

Понадобится установленный **Python 3.10 или новее** (модуль `tkinter`
входит в стандартную установку Python для Windows — отдельно ставить
ничего не надо).

1. Скопируй на другой компьютер всю папку проекта: файл `clicker.py`,
   папку `game/` и, по желанию, `icon.ico`.
2. Запусти игру: открой папку проекта в терминале и выполни
   `python clicker.py`.

## Как запустить на этом компьютере

- Готовый exe: `dist\Поднятие стримера в одиночку.exe`.
- Из терминала: `python clicker.py`.

```

├── icon.ico              иконка приложения

├── .gitignore
├── game/                 пакет с игровой логикой
│   ├── __init__.py
│   ├── state.py          состояние игры, сохранение и загрузка JSON
│   ├── upgrades.py       описание улучшений и их покупка
│   ├── mechanics.py      расчёт дохода, вирусные моменты, пассивный доход
│   ├── achievements.py   достижения и проверка их выполнения
│   ├── sound.py          синтез и воспроизведение звуков
│   ├── theme.py          палитра и шрифты тёмной темы
```

Звуковые файлы `snd_*.wav` создаются при первом запуске рядом с файлом
сохранения — они синтезируются программно (модули `wave` и `struct`),
готовых файлов в проекте нет.

## Сборка в .exe

Готовое приложение собирается в один исполняемый файл через PyInstaller:

```
python -m pip install pyinstaller
python -m PyInstaller --onefile --windowed --icon=icon.ico --add-data "icon.ico;." --name "Поднятие стримера в одиночку" clicker.py
```

Готовый `.exe` появится в папке `dist/`. Папка `build/` и файл `*.spec` —
вспомогательные, их можно удалить.

- `--onefile` — собрать всё в один файл.
- `--windowed` — не открывать окно консоли.
- `--icon=icon.ico` — иконка для самого файла exe.
- `--add-data "icon.ico;."` — вложить иконку внутрь exe, чтобы её можно
  было поставить и на окно игры.

Модуль `game/paths.py` определяет, что для собранного `.exe` сохранение и
звуки кладутся рядом с самим `.exe` (иначе прогресс терялся бы — `--onefile`
распаковывает код во временную папку, удаляемую при выходе).

## Как играть

- **ВЫЙТИ В ЭФИР** — главная кнопка, каждый стрим приносит фолловеров.
  Над ней всплывает «+N».
- **Прокачка канала:**
  - *Харизма* — больше фолловеров за каждый стрим;
  - *Нарезки клипов* — пассивный доход, фолловеры капают сами раз в секунду;
  - *Рекламная кампания* — +25% ко всем фолловерам;
  - *Вирусный момент* — шанс собрать за стрим в 5 раз больше фолловеров.
- Стоимость каждого улучшения растёт с уровнем.

## Достижения

В игре 12 достижений: за количество стримов, за набранных фолловеров, за
прокачку канала. При получении проигрывается звук и появляется уведомление.
Полный список — по кнопке **Достижения**.

## Звук

Три звука (стрим, вирусный момент, достижение) синтезируются как чистые
тоны с огибающей. Воспроизводятся на Windows через стандартный модуль
`winsound`. На других ОС звук автоматически отключается. Кнопка
**Без звука** выключает звук в любой момент, её состояние сохраняется.

## Сохранение прогресса

Прогресс хранится в файле `clicker_save.json` — при запуске из исходников
в корне проекта, у собранного `.exe` рядом с самим `.exe`:

- сохраняется автоматически при закрытии окна;
- кнопка **Сохранить** записывает прогресс вручную;
- кнопка **Сбросить** удаляет сохранение и начинает игру заново.

Если файл сохранения повреждён, игра не падает, а начинает новую игру.
